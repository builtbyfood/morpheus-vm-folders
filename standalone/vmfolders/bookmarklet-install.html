VM Folders Bookmarklet

Instant VM folder management on any Morpheus page — no install required

1

Drag this button to your bookmarks bar

Drag and drop the button below onto your browser's bookmarks bar. If the bar isn't visible, press Ctrl+Shift+B.

📁 VM Folders
↑ Drag this to your bookmarks bar
2

Get your API token

You'll need this the first time you open VM Folders.

Profile icon → User Settings → API Access → Regenerate
Copy the token. It's stored locally in your browser.
3

Click the bookmark on any Morpheus page

VM Folders slides in as a panel on the right. Click it again to hide. Your folders and VM assignments are shared with the plugin (if installed) via the same JSON database.

Works on all major browsers
ChromeBraveFirefoxEdgeSafari
